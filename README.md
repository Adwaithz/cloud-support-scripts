# Cloud Support Scripts
A collection of Bash scripts for cloud support and log analysis tasks.

## Scripts
- **log-checker.sh** — Scans log files and filters ERROR and WARNING lines
- **disk-alert.sh** — Monitors disk usage and alerts if above 80% threshold
- **service-checker.sh** — Checks if a Linux service is running or stopped
- **aws-instance-check.sh** — Lists EC2 instances and their current state via AWS CLI
